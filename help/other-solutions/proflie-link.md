# Audience Manager — 設定檔連結{#audience-manager-profile-link}

設定檔連結中的裝置圖表資料可作為合併規則選項使用。 合併規則可讓您控制Audience Manager用來劃分割槽段的資料型別。

個人資料連結可使用跨裝置資料來源，識別並收集已驗證網站訪客的特徵。 其中包括「設定檔合併規則」功能。 透過設定檔合併規則，您可以控制用於細分的資料集，並可以在多個裝置上準確地鎖定人員。

請參閱 [設定檔連結檔案](https://docs.adobe.com/content/help/en/audience-manager/user-guide/features/profile-merge-rules/merge-rules-overview.html) 以取得詳細資訊。
